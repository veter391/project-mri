/// what it analyzes

Six layers of codebase intelligence, working together.

Each analyzer extracts one slice of truth. The scoring engine combines them into explainable risk profiles. Nothing is hidden, nothing is magic.

01 · git history analyzer

Full commit history with author, message, and file changes. Per-file churn (lines added/removed over time), change frequency, ownership evolution, co-change patterns, and historical hotspots. Optionally flags large commits with generic messages that often correlate with AI tooling.

  • Churn calculation · 30/90/365-day windows
  • Co-change detection (files that move together)
  • Ownership evolution timeline
  • Optional: heuristic AI-commit flagging
02 · code structure

Multi-language AST parsing via tree-sitter. Extracts imports, call graphs, complexity metrics. Works across Python · TS/JS · Go · Rust · Java · PHP · C/C++.

  • 6+ languages supported
  • Cyclomatic + cognitive complexity
  • Dead code & duplication signals
03 · graph builder

Combines data into 4 interconnected graphs: dependency graph, call graph, evolution graph (temporal layers), and knowledge graph (files ↔ contributors).

  • NetworkX in-memory processing
  • Persisted & indexed for fast queries
04 · scoring & risk

Composite, explainable scores: architecture health, technical debt index, bus factor, knowledge islands, coupling evolution. Every number links to its contributing signals.

  • No "magic" composite scores
  • Drill down to each signal
05 · ai impact detector

Detects and quantifies AI-tool influence on your codebase via commit patterns and (optionally) ingestion of session logs from supported AI tools.

  • Heuristic + optional LLM (Ollama)
  • Tracks concentration of AI changes
06 · output layer

Four ways to consume results. CLI summary for CI, self-contained HTML report for sharing, interactive dashboard for exploration (Phase 2), and machine-readable JSON for AI agents (Phase 3 MCP).

  • CLI summary · Typer + Rich output
  • Self-contained HTML (single file, embeddable)
  • Interactive dashboard · Phase 2 (Tauri desktop)
  • JSON export · Phase 3 MCP server
